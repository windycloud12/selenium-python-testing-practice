selenium-python-testing-practice
---


Languages and Frameworks
---
- Python
- Selenium
- Pytest

Execution
---
- cd 到專案目錄下
- `python -m venv venv`　建立虛擬環境
- `venv\scripts\activate` 開啟虛擬環境 
- `pip install -r requirements.txt` 安裝依賴包
- `run_tests.bat`

Setting
---
- config.py: 設定個人參數

Output
---
- reports\report.html
- log\\*.log
